¶汇总

我们从最近的一系列分析中：

  • Pod -> Node-A
    通过 veth 设备将数据从容器的 eth0 发出
  • Node-A -> Node-B
    通过路由表将数据发送到 eth0，直接转发IP包
  • Node-B -> Pod
    eth0 使用路由表将数据从 veth 输入到 Pod 的 eth0
