¶ClusterIP 模式

按照官网的说明，ClusterIP 有三种工作模式

  • userspace(since v1.0)
  • iptables (since v1.1)
  • ipvs (since v1.9)

我们可以从 kube-proxy 的启动日志中发现：

如果没有特意的指定，我们所使用的还是iptables 的方式
