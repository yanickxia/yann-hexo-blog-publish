¶伴生对象

kotlin 去除了 static 关键字，如果我们希望声明一个 static 的函数，我们可以可以定一个 pacage-level 的函数，但是 pacage-level 不能够访问一个类的私有变量，解决方案是采用 companion 关键字进行伴生对象申明。

值得注意，你如果希望能够被 java 调用，你需要在属性或者函数上加上一个注解 @JvmStatic / @JvmField。
