¶语言学习

当然程序员应该懂好几门语言，入门的时候，我还是推荐 Python | C | Java 三者之一

面向对象 语言处理 IO 一般还是会比 函数式 语言略好的一点，但是 IO 是我们程序不可回避的部分，编写一个入门的 CLI 程序是作为入门的最好办法。
