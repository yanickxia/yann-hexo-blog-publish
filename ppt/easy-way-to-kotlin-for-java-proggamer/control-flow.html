IF代码块


				val max = if (a > b) {
					print("Choose a")
					a
				} else {
					print("Choose b")
					b
				}

`Kotlin` 和 `Java` 不同之处在于 `if` 的分支可以是代码块，`最后的表达式`作为该块的值，如示例代码中的 `a` 和 `b`。

When


				when (x) {
					1 -> print("x == 1")
					2 -> print("x == 2")
					else -> { // 注意这个块
						print("x is neither 1 nor 2")
					}
				}

`when` 将它的参数和所有的分支条件 顺序 比较，直到某个分支满足条件，如果其他分支都不满足条件将会求值 `else` 分支。

When in Range

甚至可以是在一个 `Range` 内


				var dayOfMonth = 5
				when(dayOfMonth) {
					in 1..7 -> println("We're in the first Week of the Month")
					!in 15..21 -> println("We're not in the third week of the Month")
					else -> println("none of the above")
				}
